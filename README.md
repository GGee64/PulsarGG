# PulsarGG

PulsarGG - совсем небольшой пет-проект для моей домашней Windows системы.
Мне не хватало красоты на рабочем столе, я люблю всякие визуальные приколюхи и наконец собрался и сделал с помощью локального ИИ Qwen3.6 пульсирующий эквалайзер реагирующий на любое аудио.
Программа написана за 2 полных дня, Qwen полностью ответственен за код, я только помогал ему понять в чем заключаются проблемы во время дебага,
все-таки Qwen довольно часто пытается найти проблему там где ее нет и водит пользователя за нос.

Основная функциональность на момент версии 1.0:
1. Две линии эквалайзера из 64 полос каждая, реагирующих на любое аудио, расположенные по бокам рабочего стола, не перекрываемые никакими приложениями, кроме панели задач.
2. PulsarGG висит в трее и выключается по клику левой кнопкой мыши по иконке. Так же включается обратно. Эта фича (выключение и включение по клику) - исключительно на совести ИИ и я не проверял эту часть кода.
3. PulsarGG отслеживает аудиопотоки и при переключении на другое устройство - должно его захватывать и корректно обрабатывать. Этот пункт требует тестирования на других системах на случай скрытых ошибок.
4. При отсутствии аудиопотока отыгрывает волны.
5. PulsarGG не требует прав администратора, поэтому легко работает через автозагрузку.
6. Приложение должно корректно работать в отличных от full hd разрешениях, но у меня нет возможности протестировать.
7. Приложение делалось с расчетом на Windows 10\11 64-бит системы и может не работать в более ранних версиях и\или 32-битных версиях.

На данный момент приложение не имеет настроек, возможно я займусь его усовершенствованием, но обещать ничего не могу. Фактически, текущей функциональности мне уже хватает.

--------------------------------------------------------

PulsarGG is a lightweight pet project built for my personal Windows setup. I felt my desktop was lacking some visual flair, and since I love desktop eye candy, I finally sat down and used a local AI (Qwen 3.6) to create a pulsing equalizer that reacts to any system audio. The program was developed in just two full days. Qwen handled the entire codebase; my role was strictly guiding it through debugging. Admittedly, Qwen often looked for issues where there were none and led me on a wild goose chase.

Key Features (v1.0):
1. Dual 64-band equalizer lines that react to any system audio, positioned on both sides of the desktop. They stay on top of everything except the taskbar.
2. System Tray integration: PulsarGG runs in the background and toggles on/off with a simple left-click on its tray icon. Note: This specific toggle feature was entirely generated by the AI, and I haven't fully verified this part of the code.
3. Dynamic audio routing: PulsarGG tracks active audio streams. When switching playback devices, it is designed to capture the new device and process it correctly. This feature needs more testing on different setups to catch any hidden bugs.
4. Idle animation: Plays a wave effect when no audio is playing.
5. No Admin rights required: It runs easily on startup without elevated privileges.
6. Resolution scaling: The app is designed to scale correctly on non-Full HD displays, though I haven't been able to test this myself.
7. Compatibility: Tailored for 64-bit Windows 10/11 systems. It might not work on older or 32-bit Windows versions.

Currently, the application has no configuration settings. I might improve it in the future, but I can't promise anything. Honestly, its current functionality is already more than enough for my needs.
